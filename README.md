# BAT Utils

A collection of utilities for RKeeper (RK7), StoreHouse, and other UCS software.

## Features

- Client management
- Various maintenance utilities
- Lightweight and fast
- Windows desktop application (.NET Framework 4.8)

## Requirements

- Windows 7 SP1 or newer
- .NET Framework 4.8

## Status

BAT Utils is currently under active development. New features and improvements are added regularly.
